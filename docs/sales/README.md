# Sales enablement Parrit

Date : 2026-06-24
Statut : interne

Ce dossier contient les supports sources pour transformer un rendez-vous en suite personnalisee : mail, mini-presentation, note partageable, ressource jointe, puis propale protegee quand le deal devient chaud.

## Ordre d'utilisation

1. Lire `post-meeting-value-prop-kit.md` pour choisir la personne, le niveau de maturite et la promesse principale.
2. Utiliser `post-meeting-template.md` pour rediger le mail ou la note de suite.
3. Utiliser `post-meeting-presentation-template.md` pour monter une presentation courte.
4. Piocher dans `persona-value-prop-matrix.md` si plusieurs personnes doivent etre adressees dans le meme compte.
5. Joindre une ressource source depuis `resources/` quand le prospect a besoin d'un support court a partager.

## Regle de fond

Le support parle d'abord de la personne rencontree : son quotidien, son risque, le flux a rendre plus clair. Parrit arrive ensuite comme operating partner agentique, avec un premier cas concret, un avant / apres et un prochain pas simple.
