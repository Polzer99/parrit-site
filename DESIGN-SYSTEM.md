# Parrit.ai Design System

Source de verite DA entreprise, canon valide le 04/07.

## Tokens Couleur

- `--bg: #FFFDFA` : creme chaud.
- `--alt: #F0F0F0` : surface alternative.
- `--ink: #0C0C0D` : encre principale.
- `--red: #D1132F` : accent Parrit.
- `--red-tint: rgba(209,19,47,.10)` : fond de badge.
- `--muted: #6E7079` : texte secondaire.
- `--line: #D0D8D7` : filets et bordures.
- `--terra: #C67C60` : accent rare.

## Typographie

- Geist pour les titres, `letter-spacing: -0.04em`, poids 500-600.
- Geist Mono pour corps, labels et boutons.
- Labels en uppercase.
- Polices deja chargees via `next/font` dans les layouts.

## Principe

Brutalist minimalism facon Y Combinator.

- Zero ombre portee.
- Zero bord arrondi : `border-radius: 0`.
- Zero degrade decoratif.
- Filets 1px.
- Whitespace massif.
- Cartes plates.
- Badge = fond `--red-tint`, texte `--red`, Geist Mono uppercase, angles nets.
