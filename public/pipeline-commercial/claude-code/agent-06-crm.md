# Agent 06 — Remplir le CRM tout seul

> Étape « Closing » du pipeline commercial. Brique Parrit : machine-unifiée Supabase · ingestion multi-canal.

## Ce qu'il fait pour vous

Le CRM se remplit automatiquement à partir de tous les canaux. Un seul geste enregistre l'échange, pose la prochaine action et sa date, et branche la suite. Les deals s'enrichissent sans qu'un commercial y touche.

## Le coût, sans lui

Le CRM que personne ne remplit. Les deals sans date, sans prochaine action, la donnée pourrie sur laquelle aucune décision ne tient. Le pilotage devient une devinette.

## Démarrer en 2 minutes

1. Ouvrez **Claude Code** dans un dossier de travail (voir le guide de démarrage si ce n'est pas installé).
2. Copiez le bloc ci-dessous et collez-le dans Claude Code.
3. Remplacez ce qui est entre [crochets] par vos informations, puis envoyez.

## Le prompt à coller

```
Tiens mon CRM à jour à partir de TOUS mes canaux (mails, transcripts d'appels, notes vocales), sans que j'y touche.

- À chaque échange : identifie le deal, résume ce qui s'est dit en 2 lignes, pose la prochaine action et sa date, mets à jour le montant et l'étape.
- Chaque semaine, signale-moi : les deals sans prochaine action, ceux dont la date est dépassée, ceux sans nouvelle depuis 14 jours.

Règles strictes : jamais un deal orphelin ; une entrée = une action datée ; ne modifie jamais un montant ou une étape sans la source qui le justifie.
```

## Ce que vous obtenez

Un résultat prêt à l'emploi, que vous validez avant toute action.

## La règle

L'agent prépare, **vous décidez**. Aucun envoi, aucun engagement, aucun prix n'est décidé sans vous.
