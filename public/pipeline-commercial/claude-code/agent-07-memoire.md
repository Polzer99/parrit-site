# Agent 07 — Le cerveau commun de l'équipe

> Étape « Closing » du pipeline commercial. Brique Parrit : historique partagé Supabase · jumeau PAUL_TWIN.

## Ce qu'il fait pour vous

Une mémoire commune que toute l'équipe partage et qui alimente tous les agents. Ce qu'un commercial apprend sur un compte, un secteur ou une objection, l'équipe le garde et le rejoue. Le système devient plus fort à chaque échange.

## Le coût, sans lui

Le savoir part avec le commercial qui part. Chaque nouveau réapprend ce que l'équipe savait déjà, et les mêmes erreurs se répètent d'un trimestre à l'autre.

## Démarrer en 2 minutes

1. Ouvrez **Claude Code** dans un dossier de travail (voir le guide de démarrage si ce n'est pas installé).
2. Copiez le bloc ci-dessous et collez-le dans Claude Code.
3. Remplacez ce qui est entre [crochets] par vos informations, puis envoyez.

## Le prompt à coller

```
Construis et tiens la mémoire commerciale de mon équipe dans un fichier partagé.

Ce qu'on capitalise :
- par compte : historique, décideurs, contexte, ce qui a été dit ;
- par secteur : langage, enjeux, déclencheurs qui reviennent ;
- par objection : l'objection rencontrée + la réponse qui a marché.

Avant chaque message ou chaque call, relis la mémoire du compte ET du secteur, et applique ce qu'on sait déjà. Après chaque échange, ajoute la leçon : ce qui a marché, ce qui a coincé. Ce qu'un commercial apprend, toute l'équipe le rejoue.
```

## Ce que vous obtenez

Un résultat prêt à l'emploi, que vous validez avant toute action.

## La règle

L'agent prépare, **vous décidez**. Aucun envoi, aucun engagement, aucun prix n'est décidé sans vous.
